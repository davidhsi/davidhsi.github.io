Breast Cancer Tumor Classification Project

Problem Statement: Breast Cancer is the most common type of cancer for women worldwide. Tumors form when cells in the breast tissue proliferate uncontrollably. These tumors can be either malignant (harmful) or benign (harmless). We wanted to create a model to accurately classify breast tumors into malignant or benign based on physiological features (such as radius, texture, etc.) to optimize the treatment process.

  • Collaborated with 3 other students to create advanced linear and nonlinear Python models in Jupyter Notebooks
  • Logistic regression and an unpruned decision tree were used as the base models
  • Went through pruned decision tree, bagging, random forest, XGboost, and ensemble methods to identify the best model that gave the highest Accuracy, lowest False Negative Rate, and highest Precision
  • The best model identified was the XGBoost model and the single decision tree giving an Accuracy of 97.5%, False Negative Rate of 16.7%, and Precision of 83.3%.
